System Requirements
-------------------

Python 3.4 - If your running Arch, this shouldn't be a problem!

- simplekml

If you want to be able to send the GeoJSON to geojson.io via a GitHub gist:

- github3.py
- `geojsonio.py <https://github.com/jwass/geojsonio.py>`_
