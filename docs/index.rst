ArchMap
=======

The ArchMap project creates a map of Arch Linux users all over the world.

* Project page on the ArchWiki: https://wiki.archlinux.org/index.php/ArchMap

* The list of user data: https://wiki.archlinux.org/index.php/ArchMap/List

* Project talk on the Arch forums https://bbs.archlinux.org/viewtopic.php?id=22518

* Repository on GitHub: https://github.com/maelstrom59/ArchMap
